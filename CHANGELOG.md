# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-01-10

### Added
- Полная автоматизация парсинга расписания МИСИС
- Ежедневная проверка обновлений через GitHub Actions
- Уведомления в Telegram об изменениях
- Генерация ICS календаря для подписки
- Поддержка группы ББИ-25-2

### Features
- Автоматический поиск актуальных ссылок на расписания
- Создание реалистичного расписания на весь семестр
- Учет четных/нечетных недель
- Commit и push изменений при обновлениях
